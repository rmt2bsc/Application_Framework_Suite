Contains classes and interfaces for routing messages internally to their destination without knowing the actual messaging architecture. These routers use the ServiceResistry to determine their appropriate destinations.
How to use
	      To route a SOAP message

	      SOAPMessage sm = MessageFactory.newInstance().createMessage();
	      String messageId = "RQ_Get_Customer";
	      MessageRouterHelper helper = new MessageRouterHelper();
	      SOAPMessage results = helper.routeSoapMessage(messageId, sm);
	      
	      To route a JAXB message

	      JaxbObject obj = ObjectFactory.createJaxbObject();
	      String messageId = "RQ_Get_Customer";
	      MessageRouterHelper helper = new MessageRouterHelper();
	      Object results = helper.routeJaxbMessage(messageId, obj);
